# BIM AI Generator - Multi-Agent Architecture

A professional AI-powered Building Information Modeling (BIM) system that uses multiple specialized agents to generate high-quality IFC models.

## 🏗️ Architecture Overview

This project implements a **multi-agent system** where specialized AI agents collaborate to generate, validate, and review BIM code:

1. **Orchestrator Agent** - Understands requirements and coordinates workflow
2. **Code Generator Agent** - Generates BlenderBIM Python code  
3. **Code Validator Agent** - Validates and refines generated code
4. **Architecture Specialist Agent** - Reviews complex structures for design quality

See [MULTI_AGENT_ARCHITECTURE.md](./MULTI_AGENT_ARCHITECTURE.md) for detailed documentation.

## ✨ Features

- 🤖 **Multi-agent AI system** for robust code generation
- 🏢 **Complex BIM models** - Generate complete buildings, bridges, structures
- ✅ **Automatic validation** - Code is validated before execution
- 🎨 **Material & color support** - Apply realistic materials and colors
- 🔍 **Architecture review** - Professional design validation
- 💬 **Conversational interface** - Natural language interaction
- 📦 **IFC export** - Industry-standard file format
- 🖼️ **3D visualization** - Real-time model preview
